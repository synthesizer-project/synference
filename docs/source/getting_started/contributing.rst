Getting Help & Contributing
===========================

Help
~~~~

If you wish to report an issue or get help with Synference, please `raise an issue <https://github.com/synthesizer-project/synference/issues>`_ on Github.

We also have a dedicated slack group for updates and general help with synthesizer and synference. All are welcome to join; please email <chris dot lovell dot astro at gmail dot com> for a sign up link.
